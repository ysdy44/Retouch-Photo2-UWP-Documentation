# **RGB**
![Image](Images/Menus_ColorMenu_RGB.jpg)

|**按钮**|**介绍**|**范围**|
|:-|:-|:-|
|**R**|**红色通道**|**( 0 < R < 255 )**|
|**G**|**绿色通道**|**( 0 < G < 255 )**|
|**B**|**蓝色通道**|**( 0 < B < 255 )**|


---
<br/>

# **HSV**
![Image](Images/Menus_ColorMenu_HSV.jpg)

|**按钮**|**介绍**|**范围**|
|:-|:-|:-|
|**H**|**色相分量**|**( 0° < H < 360° )**|
|**S**|**饱和度分量**|**( 0% < S < 100% )**|
|**V**|**明度分量**|**( 0% < V < 100% )**|
